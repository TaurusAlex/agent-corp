# Video Script Examples

## Example 1: Product Launch (60s)
```
Title: [Product] - Finally Here!
Duration: 60s

[0:00-0:03] HOOK
Visual: Product packaging reveal
Audio: "Wait for it..."

[0:03-0:10] PROBLEM
Visual: Person frustrated, scrolling
Audio: "You've been searching for [X] for months..."

[0:10-0:30] SOLUTION
Visual: Product in use, happy reactions
Audio: "Introducing [Product] - with [key feature]..."

[0:30-0:45] PROOF
Visual: Testimonials, ratings
Audio: "Loved by 10,000+ customers..."

[0:45-0:60] CTA
Visual: Product hero, discount code
Audio: "Get yours now - link in bio!"
```

## Example 2: Tutorial (90s)
```
Title: How to [Desired Outcome] in 3 Steps
Duration: 90s

[0:00-0:05] HOOK - Before/after split
[0:05-0:30] STEP 1 - Screen demo
[0:30-0:55] STEP 2 - Continued demo
[0:55-1:20] STEP 3 - Final step
[1:20-1:30] CTA - Link below
```
