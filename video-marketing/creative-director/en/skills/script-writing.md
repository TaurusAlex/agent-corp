# Script Writing

## When to Use
- Creating marketing video scripts (15s, 30s, 60s, 2min+)
- Writing product launch videos
- Developing explainer video content
- Crafting tutorial/instructional videos
- Any video that needs a spoken narrative

## Workflow

### Step 1: SCQA Framework
Ask/get:
- **S**ituation: What's the product/service?
- **C**omplication: What problem does it solve?
- **Q**uestion: What's the key message?
- **A**nswer: What's the call-to-action?

### Step 2: Structure the Script
Standard video structure:
- **Hook (0-3s)**: Attention grabber - surprising fact, question, or emotion
- **Problem (3-10s)**: Establish the pain point
- **Solution (10-30s)**: Present the product/value
- **Proof (30-45s)**: Social proof, testimonials, data
- **CTA (last 3-5s)**: Clear action to take

### Step 3: Flesh Out
For each section:
- Exact words to say
- Visual suggestions
- Audio/music cues
- Timing estimates

### Step 4: Refine
- Check total timing
- Ensure message clarity
- Verify CTA is compelling
- Adjust for platform (TikTok vs YouTube)

## Output Format
```
Title: [Video Title]
Duration: [X] seconds

[0:00-0:03] HOOK
Visual: [What viewers see]
Audio: [What they hear]

[0:03-0:10] PROBLEM
...

[CTA] Include specific action and any offer code
```

## Examples

### Product Launch (60s)
```
Hook: "Wait for it..."
Problem: "You've been searching for X for months..."
Solution: "Introducing [Product] with [key feature]..."
Proof: "Loved by 10,000+ customers..."
CTA: "Get yours now - link in bio!"
```

### Tutorial (90s)
```
Hook: Before/after split screen
Step 1: [Demo first step]
Step 2: [Demo second step]  
Step 3: [Demo final step]
CTA: "Try it now - link below!"
```

## Tips
- Lead with benefit, not feature
- "You" vs "we" language
- One clear CTA per video
- Match tone to platform culture
