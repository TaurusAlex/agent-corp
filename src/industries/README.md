# Agent.Corp 行业分类

基于 GB/T 4754-2017 国民经济行业分类标准

## 目录结构

```
src/
├── industries/
│   ├── primary/        # 第一产业 (农、林、牧、渔业)
│   ├── secondary/      # 第二产业 (工业)
│   └── tertiary/       # 第三产业 (服务业)
```

## 行业分类

- **第一产业**: 农、林、牧、渔业 (门类A)
- **第二产业**: 采矿业、制造业、电力热力燃气及水生产和供应业、建筑业 (门类B-F)
- **第三产业**: 批发和零售业、交通运输仓储和邮政业、住宿和餐饮业等 (门类G-T)

## 行业代码说明

- 第1位: 产业 (1-第一产业, 2-第二产业, 3-第三产业)
- 第2位: 门类 (A-T)
- 第3-4位: 大类 (01-97)
- 第5位: 中类 (0-9)
- 第6位: 小类 (0-9)

## 使用方法

每个小类目录下包含:
- `agent.md`: AI Agent 配置模板
