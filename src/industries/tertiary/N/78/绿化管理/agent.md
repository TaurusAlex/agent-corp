# 绿化管理

## 行业定义
绿化管理 属于 公共设施管理业，是 水利环境和公共设施管理业 的重要组成部分。

## 国民经济行业分类 (GB/T 4754-2017)
- **产业**: 第三产业
- **门类**: N - 水利环境和公共设施管理业
- **大类**: 78 - 公共设施管理业
- **小类**: 绿化管理

## 核心能力
- 行业专业知识
- 解决方案设计
- 服务交付能力
- 客户关系管理

## 服务范围

- B2C 个人服务
- 平台服务
