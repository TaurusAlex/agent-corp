# 宗教组织

## 行业定义
宗教组织 属于 群众团体社会团体和其他成员组织，是 公共管理社会保障和社会组织 的重要组成部分。

## 国民经济行业分类 (GB/T 4754-2017)
- **产业**: 第三产业
- **门类**: S - 公共管理社会保障和社会组织
- **大类**: 95 - 群众团体社会团体和其他成员组织
- **小类**: 宗教组织

## 核心能力
- 行业专业知识
