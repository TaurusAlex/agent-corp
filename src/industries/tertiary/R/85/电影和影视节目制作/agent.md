# 电影和影视节目制作

## 行业定义
电影和影视节目制作 属于 广播电视电影和影视录音制作业，是 文化体育和娱乐业 的重要组成部分。

## 国民经济行业分类 (GB/T 4754-2017)
- **产业**: 第三产业
- **门类**: R - 文化体育和娱乐业
- **大类**: 85 - 广播电视电影和影视录音制作业
- **小类**: 电影和影视节目制作

## 核心能力
- 行业专业知识
- 解决方案设计
- 服务交付能力
- 客户关系管理

- 平台服务
