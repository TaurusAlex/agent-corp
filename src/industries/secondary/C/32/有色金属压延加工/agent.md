# 有色金属压延加工

## 行业定义
有色金属压延加工 属于 有色金属冶炼和压延加工业，是 制造业 的重要组成部分。

## 国民经济行业分类 (GB/T 4754-2017)
- **产业**: 第二产业
- **门类**: C - 制造业
- **大类**: 32 - 有色金属冶炼和压延加工业
- **小类**: 有色金属压延加工
