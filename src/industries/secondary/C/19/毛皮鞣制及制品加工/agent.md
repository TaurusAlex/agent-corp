# 毛皮鞣制及制品加工

## 行业定义
毛皮鞣制及制品加工 属于 皮革毛皮羽毛及其制品和制鞋业，是 制造业 的重要组成部分。

## 国民经济行业分类 (GB/T 4754-2017)
- **产业**: 第二产业
- **门类**: C - 制造业
- **大类**: 19 - 皮革毛皮羽毛及其制品和制鞋业
- **小类**: 毛皮鞣制及制品加工
